# Levi's Road to Black 🏉

A rugby development app for Levi — training, nutrition, the laws of the game, growth tracking,
and the pathway from club rugby to the All Blacks.

Single-file PWA. No build step, no dependencies. Works offline, installable to a phone home screen.

- **Home** — level, XP, streak, this block's focus, days to game day
- **Train** — skill checklist (handling, contact, running, kicking, game sense, athletic development)
- **Fuel** — daily nutrition habits + the good-plate guide
- **Rules** — two-track laws quiz (Small Blacks U11 grade + the national game)
- **Progress** — achievements, game log, growth charts, milestones
- **Pathway** — the 11 rungs from Southern club, Hinds, to the All Blacks

Built with Dad. Bred in, not managed in.
